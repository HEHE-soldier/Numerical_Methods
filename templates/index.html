Jacobi Iteration Method

Mathematical Discussion

The Jacobi method is an iterative algorithm for determining the solutions of a strictly diagonally dominant system of linear equations. Each diagonal element is solved for, and an approximate value is plugged in.

For a system $Ax = b$, the iteration is defined by:

$$x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j \neq i} a_{ij} x_j^{(k)} \right)$$

Worked Examples

Example 1

Example 1: 2x2 System

Consider the diagonally dominant system:

$$4x_1 + x_2 = 5$$

$$x_1 + 3x_2 = 4$$

First, isolate $x_1$ and $x_2$ on the left side:

$$x_1^{(k+1)} = \frac{1}{4}(5 - x_2^{(k)})$$

$$x_2^{(k+1)} = \frac{1}{3}(4 - x_1^{(k)})$$

We start with an initial guess of $x_1^{(0)} = 0$ and $x_2^{(0)} = 0$.

Iteration 1 ($k=0$):

$$x_1^{(1)} = \frac{1}{4}(5 - 0) = 1.25$$

$$x_2^{(1)} = \frac{1}{3}(4 - 0) \approx 1.3333$$

Iteration 2 ($k=1$):

$$x_1^{(2)} = \frac{1}{4}(5 - 1.3333) = 0.9167$$

$$x_2^{(2)} = \frac{1}{3}(4 - 1.25) = 0.9167$$

Iteration 3 ($k=2$):

$$x_1^{(3)} = \frac{1}{4}(5 - 0.9167) \approx 1.0208$$

$$x_2^{(3)} = \frac{1}{3}(4 - 0.9167) \approx 1.0278$$

The values are converging toward the exact solution of $x_1 = 1$, $x_2 = 1$.

Example 2: 3x3 System

Consider the system:

$$5x_1 - x_2 + x_3 = 10$$

$$2x_1 + 8x_2 - x_3 = 11$$

$$-x_1 + x_2 + 4x_3 = 3$$

Isolating the variables yields our iteration formulas:

$$x_1^{(k+1)} = \frac{1}{5}(10 + x_2^{(k)} - x_3^{(k)})$$

$$x_2^{(k+1)} = \frac{1}{8}(11 - 2x_1^{(k)} + x_3^{(k)})$$

$$x_3^{(k+1)} = \frac{1}{4}(3 + x_1^{(k)} - x_2^{(k)})$$

Using an initial guess of $x_1^{(0)} = 0$, $x_2^{(0)} = 0$, $x_3^{(0)} = 0$.

Iteration 1 ($k=0$):

$$x_1^{(1)} = \frac{1}{5}(10 + 0 - 0) = 2.0$$

$$x_2^{(1)} = \frac{1}{8}(11 - 2(0) + 0) = 1.375$$

$$x_3^{(1)} = \frac{1}{4}(3 + 0 - 0) = 0.75$$

Iteration 2 ($k=1$):

$$x_1^{(2)} = \frac{1}{5}(10 + 1.375 - 0.75) = 2.125$$

$$x_2^{(2)} = \frac{1}{8}(11 - 2(2.0) + 0.75) = 0.96875$$

$$x_3^{(2)} = \frac{1}{4}(3 + 2.0 - 1.375) = 0.90625$$

This rapidly approaches the exact solution of $x_1 = 2$, $x_2 = 1$, $x_3 = 1$.
