Jacobi Iteration Method

Mathematical Discussion

The Jacobi method is an iterative algorithm used to approximate the solution to a system of equations.

Worked Examples

Example 1: 2x2 System

Consider the functions:

$$4x_1 + x_2 = 5$$

$$x_1 + 3x_2 = 4$$

First, isolate x_1 and x_2 on the left side:

$$x_1 = \frac{1}{4}(5 - x_2)$$

$$x_2 = \frac{1}{3}(4 - x_1)$$

We start with an initial guess of x_1 = 0 and x_2 = 0.

Iteration 1 x_1 = 0, x_2 = 0:

$$x_1 = \frac{1}{4}(5 - 0) = 1.25$$

$$x_2 = \frac{1}{3}(4 - 0) = 1.3333$$

Iteration 2 x_1 = 1.25 and x_2 = 1.3333:

$$x_1 = \frac{1}{4}(5 - 1.3333) = 0.9167$$

$$x_2 = \frac{1}{3}(4 - 1.25) = 0.9167$$

Iteration 3 x_1 = 0.9167 and x_2 = 0.9167:

$$x_1 = \frac{1}{4}(5 - 0.9167) = 1.0208$$

$$x_2 = \frac{1}{3}(4 - 0.9167) = 1.0278$$

Iteration 4 x_1 = 1.0208 and x_2 = 1.0278:

$$x_1 = \frac{1}{4}(5 - 1.0278) = 0.99305$$

$$x_2 = \frac{1}{3}(4 - 1.0208) = 0.99307$$

Iteration 5 x_1 = 0.99305 and x_2 = 0.99307:

$$x_1 = \frac{1}{4}(5 - 0.99307) = 1.00173$$

$$x_2 = \frac{1}{3}(4 - 0.99305) = 1.00232$$

Based on the pattern of the iterations both variables x_1 and x_2 are approaching 1, hence it is safe to conclude that $$x_1 = 1$$ and $$x_2 = 1$$.

Example 2: 3x3 System

Consider the functions:

$$5x_1 - x_2 + x_3 = 10$$

$$2x_1 + 8x_2 - x_3 = 11$$

$$-x_1 + x_2 + 4x_3 = 3$$

First, isolate x_1, x_2, and x_3 on the left side:

$$x_1 = \frac{1}{5}(10 + x_2 - x_3)$$

$$x_2 = \frac{1}{8}(11 - 2x_1 + x_3)$$

$$x_3 = \frac{1}{4}(3 + x_1 - x_2)$$

We start with an initial guess of x_1 = 0, x_2 = 0, and x_3 = 0.

Iteration 1 x_1 = 0, x_2 = 0, x_3 = 0:

$$x_1 = \frac{1}{5}(10 + 0 - 0) = 2.0$$

$$x_2 = \frac{1}{8}(11 - 2(0) + 0) = 1.375$$

$$x_3 = \frac{1}{4}(3 + 0 - 0) = 0.75$$

Iteration 2 x_1 = 2.0, x_2 = 1.375, x_3 = 0.75:

$$x_1 = \frac{1}{5}(10 + 1.375 - 0.75) = 2.125$$

$$x_2 = \frac{1}{8}(11 - 2(2.0) + 0.75) = 0.96875$$

$$x_3 = \frac{1}{4}(3 + 2.0 - 1.375) = 0.90625$$

Iteration 3 x_1 = 2.125, x_2 = 0.96875, x_3 = 0.90625:

$$x_1 = \frac{1}{5}(10 + 0.96875 - 0.90625) = 2.0125$$

$$x_2 = \frac{1}{8}(11 - 2(2.125) + 0.90625) = 0.95703$$

$$x_3 = \frac{1}{4}(3 + 2.125 - 0.96875) = 1.03906$$

Iteration 4 x_1 = 2.0125, x_2 = 0.95703, x_3 = 1.03906:

$$x_1 = \frac{1}{5}(10 + 0.95703 - 1.03906) = 1.98359$$

$$x_2 = \frac{1}{8}(11 - 2(2.0125) + 1.03906) = 1.00176$$

$$x_3 = \frac{1}{4}(3 + 2.0125 - 0.95703) = 1.01387$$

Iteration 5 x_1 = 1.98359, x_2 = 1.00176, x_3 = 1.01387:

$$x_1 = \frac{1}{5}(10 + 1.00176 - 1.01387) = 1.99758$$

$$x_2 = \frac{1}{8}(11 - 2(1.98359) + 1.01387) = 1.00584$$

$$x_3 = \frac{1}{4}(3 + 1.98359 - 1.00176) = 0.99546$$

Based on the pattern of the iterations the variables x_1, x_2, and x_3 are approaching 2, 1, and 1, hence it is safe to conclude that $$x_1 = 2$$, $$x_2 = 1$$, and $$x_3 = 1$$.
